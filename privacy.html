Privacy Policy

Last updated: July 2026

This application ("MindBites Shorts Automation") is a personal, single-user automation tool used to generate and upload short-form video content to a single YouTube channel owned and operated by the developer.

What data this app accesses

This app uses the YouTube Data API solely to upload videos to the developer's own YouTube channel. It does not access, read, or store any data belonging to other users, and it is not used by, or made available to, the general public.

Data storage

Authentication tokens are stored securely as encrypted secrets within a private GitHub repository and are never shared with, or accessible to, any third party.

Data sharing

No data collected or accessed by this application is shared with any third party for any purpose, including advertising or marketing.

Contact

For any questions regarding this privacy policy, please contact the developer through the associated GitHub repository.
